# Edit files

## Windows

### Visual Studo code

Executable: "C:\Program Files\Microsoft VS Code\bin\code.cmd"

Call: Code.cmd

### Notepad++

Executable: "C:\Program Files\Notepad++\notepad++.exe"

Call: NEdit.cmd

### Notepad

Executable: "C:\Windows\notepad.exe" (default in Windows)

Call: Nodepad.exe
